# MVP Copy Direction

## Confirmed Direction

- Use a restrained and warm tone.
- Keep the primary button copy as "帮我捋一捋" for the first prototype.
- Keep these expressions:
  - "先不用想清楚"
  - "想到什么都可以先放在这里。"
  - "也许可以先看这个"
  - "其他想法都还在"
  - "看一下"
  - "先不管"

## Rationale

The copy should feel close and natural without becoming overly comforting or pushy.
